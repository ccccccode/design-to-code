# Outputs README

这个目录由 design-to-code skill 生成。

## 目录说明

```txt
outputs/
  01-input/
    input-map.md
  02-analysis/
    page-analysis.md
  03-manifest/
    asset-manifest.json
    prompt-groups.json
  04-prompts/
    gpt-image-prompts.md
  05-generated-assets/
  06-frontend-handoff/
    frontend-handoff.md
```

## 使用方式

1. 先阅读 `02-analysis/page-analysis.md`
2. 确认 `03-manifest/asset-manifest.json`
3. 使用 `04-prompts/gpt-image-prompts.md` 生成图片
4. 把图片放到 `05-generated-assets/`
5. 使用 `06-frontend-handoff/frontend-handoff.md` 继续前端开发
